# 用户测试脚本

这份脚本用于主持人现场读给测试用户。  
建议用韩语执行，中文部分给组内理解。

## 开场说明

### KR

안녕하세요. 저희는 템플릿 기반 Multi-Agent Workflow Builder MVP를 테스트하고 있습니다.

이 시스템은 단순 챗봇이 아니라, 사용자가 선택한 템플릿에 따라 여러 기능 노드가 순서대로 실행되고, 그 결과를 조합해서 답변을 생성하는 시스템입니다.

오늘 테스트의 목적은 사용자가 이 시스템을 쉽게 이해하고 사용할 수 있는지 확인하는 것입니다. 정답을 맞히는 테스트가 아니므로, 불편하거나 이해가 안 되는 부분이 있으면 그대로 말씀해 주세요.

### ZH

你好，我们正在测试一个基于模板的 Multi-Agent Workflow Builder MVP。

这个系统不是普通聊天机器人，而是根据用户选择的模板，让多个功能节点按顺序执行，并组合结果生成最终回答。

今天不是考你答对不对，而是看系统是否容易理解、是否好用。如果有看不懂或不方便的地方，请直接说出来。

## Task A: 穿搭推荐

### KR

첫 번째 작업입니다.

Outfit Recommendation 또는 의류 추천 템플릿을 선택한 뒤, 아래 문장을 입력하고 실행해 주세요.

```text
다음 주 칭다오 여행 가는데, 나한테 맞는 옷을 추천해줘.
```

실행 후에는 최종 추천 결과와 Workflow Trace를 보고, 어떤 정보가 사용되었는지 설명해 주세요.

### ZH

第一个任务：

请选择穿搭推荐模板，输入：

```text
下周去青岛旅游，帮我推荐适合我的衣服。
```

运行后，看最终结果和 Workflow Trace，并说明系统用了哪些信息。

## Task B: 通勤穿搭

### KR

두 번째 작업입니다.

Commute Outfit 템플릿을 선택한 뒤, 아래 문장을 입력하고 실행해 주세요.

```text
내일 서울에 출근하는데, 포멀한 느낌의 옷을 추천해줘.
```

실행 후 첫 번째 작업과 어떤 점이 비슷하고 어떤 점이 다른지 말해 주세요.

### ZH

第二个任务：

请选择通勤穿搭模板，输入：

```text
明天在首尔上班，帮我推荐正式一点的衣服。
```

运行后，说一下它和第一个任务哪里相同、哪里不同。

## Task C: 非推荐模板

### KR

세 번째 작업입니다.

Presentation Planning 또는 Customer Support Ticket 템플릿 중 하나를 선택해서 실행해 주세요.

실행 후, 이 템플릿과 의류 추천 템플릿이 공통적으로 어떤 구조를 가지고 있는지 설명해 주세요.

### ZH

第三个任务：

请选择发表规划或客服工单模板运行一次。

运行后，说明它和穿搭推荐模板在结构上有什么共同点。

## 结束访谈

### KR

마지막으로 세 가지를 질문드리겠습니다.

1. 이 시스템이 무엇을 하는지 이해하기 쉬웠나요?
2. Workflow Trace가 도움이 되었나요?
3. 이 시스템이 일반 챗봇과 다르다고 느꼈나요? 다르다면 어떤 점이 달랐나요?

### ZH

最后问三个问题：

1. 你容易理解这个系统在做什么吗？
2. Workflow Trace 对你有帮助吗？
3. 你觉得它和普通聊天机器人不同吗？如果不同，哪里不同？
